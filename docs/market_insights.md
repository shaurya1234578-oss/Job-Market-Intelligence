# Market Insights

## Executive Recommendations

- Prioritize Python, SQL, Power BI, and Excel as the baseline analytics skill stack.
- Use role-specific skill clusters to decide whether to target BI analyst, data analyst, or junior data scientist roles.
- Compare salary ranges by role family and location before judging market opportunity.
- Build project proof around the highest-frequency skills instead of listing tools only.

## KPI Layer

- Posting count by role
- Posting count by location
- Skill frequency
- Salary band by role
- Skill and salary relationship
